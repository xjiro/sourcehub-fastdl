# Sourcehub FastDL

This repository contains the FastDL setup used by [Sourcehub](https://sourcehub.lol/) game servers to run Half-Life mods.

## Usage

This repository serves as the source-of-truth for Sourcehub's Half-Life servers, mod content, and custom assets. Players connecting to Sourcehub servers will automatically download necessary files from this repository.

Make pull requests for new mods and custom content.

## About Sourcehub

[Sourcehub](https://sourcehub.lol/) is a free platform hosting ephemeral Half-Life mod servers.
